# Contributing to CreamCollar

We welcome contributions to this project! Please follow these guidelines to ensure a smooth collaboration.

## Getting Started

1.  Fork the repository.
2.  Clone your forked repository: `git clone https://github.com/Creamcollar/autosar-arxml-codegen.git`
3.  Create a new branch for your feature or bug fix: `git checkout -b feature/your-feature-name` or `git checkout -b bugfix/your-bug-fix`


## Code Style

Please adhere to the existing code style and formatting.

## Submitting Changes

1.  Commit your changes with clear and concise commit messages.
2.  Push your branch to your forked repository.
3.  Open a pull request to the `main` branch of the original repository.
